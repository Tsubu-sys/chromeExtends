background.js

TTS Player の Service Worker（バックグラウンドスクリプト）。 コンテキストメニューの登録・クリック処理、TTS 再生制御、 Offscreen ドキュメントの管理、Content Script との通信を担う。

content_script.js

TTS Player の Content Script。 Shadow DOM を使ってページに UI を注入し、background.js からのメッセージに応じて 再生状態の表示・ハイライト・ドラッグなどの操作を担う。

offscreen.js

Service Worker のスリープを防ぐための Offscreen ドキュメント。 定期的にハートビートメッセージを Service Worker へ送信することで、 TTS 再生中に Service Worker が終了しないようにする。
