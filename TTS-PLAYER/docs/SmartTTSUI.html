Class: SmartTTSUI

SmartTTSUI()

TTS Player の UI を管理するクラス。 Shadow DOM を使ってホストページの CSS から隔離されたパネルを生成し、 background.js とのメッセージ通信および DOM 操作を担う。

Constructor

new SmartTTSUI()

Shadow DOM を構築し、UI 要素の描画とイベントリスナーの設定を行う。 すでに document.body が存在する場合は即時追加、未存在の場合は DOMContentLoaded を待つ。
